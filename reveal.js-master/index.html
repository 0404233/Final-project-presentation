Технологический стек

Основные библиотеки и фреймворки:

  • React: Библиотека для создания пользовательских интерфейсов.
  • React DOM: Связующее звено между React и DOM.
  • React Hook Form: Библиотека для управления формами в React.
  • React Router DOM: Для маршрутизации в приложении.
  • React Toastify: Библиотека для отображения уведомлений.
  • Swiper: Мощный слайдер и карусель.

Инструменты разработки:

  • TypeScript: Язык программирования, расширяющий JavaScript типизацией.
  • Vite: Быстрый сборщик для разработки с использованием модулей ECMAScript.
  • Vitest: Тестовый фреймворк, оптимизированный для Vite.
  • Sass: Препроцессор CSS для более удобного написания стилей.
  • Husky: Инструмент для управления Git хуками.

Линтинг и форматирование:

  • ESLint: Инструмент для анализа кода, обеспечивающий его качество.
  • Prettier: Код-форматтер для поддержания единого стиля кода.
